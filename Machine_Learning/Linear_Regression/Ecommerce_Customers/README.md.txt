# 📦 E-commerce Customers Analysis

Este projeto analisa dados de clientes de e-commerce com o objetivo de prever quanto um cliente gastará com base em características como tempo no site, tempo no app, afiliados, etc.

## 📊 Dataset

- Fonte: [Kaggle - Ecommerce Customers Dataset](https://www.kaggle.com/datasets/srolka/ecommerce-customers)

## 🔍 Etapas do Projeto

1. Exploração de Dados (EDA)
2. Análise de Correlação
3. Pré-processamento e Escalonamento
4. Treinamento de modelos de Regressão Linear e Random Forest
5. Avaliação com métricas (MSE, RMSE, R²)
6. Conclusões
