# 🌼 Projeto: Análise Univariada – Dataset Iris

Este projeto realiza uma análise univariada das variáveis numéricas do dataset clássico Iris, com foco em visualização de dados para explorar distribuições, tendências centrais, dispersão e outliers.

## 📌 Objetivos

- Explorar a distribuição de cada variável individualmente
- Utilizar visualizações claras e esteticamente agradáveis
- Desenvolver interpretações escritas profissionais baseadas nos gráficos
- Criar uma base sólida para análises futuras (bivariadas e multivariadas)

## 🗂️ Arquivo principal

- `01_univariate_analysis.ipynb`: Notebook com toda a análise passo a passo, incluindo:
  - Histogramas
  - Boxplots
  - Interpretações visuais para cada variável

## 🧰 Tecnologias utilizadas

- Python 3.x
- pandas
- matplotlib
- seaborn
- scikit-learn (para carregamento do dataset)
