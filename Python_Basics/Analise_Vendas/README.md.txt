# 📊 Análise de Vendas de Produtos - 2024

Análise profissional do desempenho de vendas de diferentes produtos ao longo de 2024.

## 🎯 Objetivos

- Analisar o volume de vendas por produto e categoria
- Identificar padrões temporais e sazonais
- Descobrir os produtos mais e menos lucrativos
- Gerar insights acionáveis para o negócio

## 🛠️ Tecnologias Utilizadas

- Python 3.8+
- Pandas
- Matplotlib/Seaborn
- Plotly
- Jupyter Notebook

## 📋 Metodologia

1. **Coleta e Pré-processamento**: Limpeza e transformação dos dados
2. **Análise Exploratória**: Estatísticas descritivas e visualizações
3. **Análise Temporal**: Identificação de padrões sazonais
4. **Análise por Produto**: Desempenho individual dos produtos
5. **Geração de Insights**: Recomendações baseadas em dados
